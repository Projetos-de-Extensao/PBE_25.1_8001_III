Como Baixar e Configurar a Aplicação

Pré-requisitos

Antes de iniciar, certifique-se de ter os seguintes softwares instalados:

  • Python (versão mais recente recomendada)
  • Django

Passos para Configuração

  1. Baixar e instalar o Python
  2. Acesse o site oficial do Python.
  3. Baixe e instale a versão mais recente.
  4. Durante a instalação, marque a opção "Add Python to PATH".

  5. Para verificar se a instalação foi concluída, abra o terminal e digite: sh python --version

  6. Criar um ambiente virtual

  7. No diretório do código, abra o terminal e execute: sh python -m venv env

  8. Ative o ambiente virtual:

    • No Windows: sh env\Scripts\activate
    • No macOS/Linux: sh source env/bin/activate

  9. Instalar o Django

  10. Com o ambiente virtual ativado, instale o Django com: sh pip install django

  11. Para verificar a instalação, execute: sh django-admin --version

  12. Instalar o MkDocs

  13. Ainda com o ambiente virtual ativado, instale o MkDocs: sh pip install mkdocs
  14. Para testar a instalação, execute: sh mkdocs --version

Agora sua aplicação está pronta para ser configurada e executada!
